## Image Requirements

- Supported formats: JPG, PNG, GIF
- Recommended size: 800x600 pixels or higher
- File size: Keep under 2MB for better performance
- Use clear, well-lit photos of the plants
- `sweet_basil.jpg`
- `green_mint.png` 
- `red_rose.jpg`
